# vue-hw

## วิธีใช้งาน
```
1.เพิ่มรายการ
    -กรอกรายละเอียด 
    -เลือกวันที่
    -เลือกประเภท(รายรับ,ราบจ่าย)
    -กรอกจำนวนเงิน(ถ้ากรอกข้อมูลเป็นค่า- ระบบจะแปลงเป็นค่า+)
    -กดAdd เพื่อเพิ่มข้อมูล
2.ตาราง
    -สามารถดูข้อมูลบันทึกรายรับ-รายจ่ายได้ในตาราง
    -สามารถดูข้อมูลรวมรายรับ, รายจ่าย, คงเหลือ ได้ที่ท้ายตาราง
3.กราฟเปรียบเทียบรายรับรายจ่าย
    -สามารถเลือกช่วงเวลาที่ต้องการดูการเปรียบเทียบรายรับ-รายจ่ายได้ด้านล่างกราฟ
    -ถ้าไม่เลือกแล้วกด Ok จะเป็นการดูการเปรียบเทียบรายรับ-รายจ่ายทั้งหมด
    -ถ้าเลือกแต่วันแรก จะเป็นการดูการเปรียบเทียบรายรับ-รายจ่ายตั้งแต่วันแรกที่เลือกเป็นต้นไป
    -ถ้าเลือกแต่วันสุดท้าย จะเป็นการดูการเปรียบเทียบรายรับ-รายจ่ายทั้งหมดจนถึงวันสุดท้ายที่เลือก
    -ถ้าเลือกทั้งวันแรกถึงวันสุดท้าย จะเป็นการดูการเปรียบเทียบรายรับ-รายจ่ายตั้งแต่วันแรกที่เลือกถึงวันสุดท้ายที่เลือก
    -เมื่อเลือกเสร็จข้อมูลจะแสดงขึ้นบนกราฟ
```
